Target Selector Rule

Values may include expressions that reference fields in the User or target object e.g.:

  • ${user.name} - the logged in user's name
  • ${user.role} - the logged in user's role
  • ${user[user.defined.field]} - the value of the specified user-defined field

These will be substituted for the referenced value to create the final value that this rule must match. The full list of fields can be found by consulting the REST API object model documentation
